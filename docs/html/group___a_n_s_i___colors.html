Detailed Description

Definizioni di codici ANSI per colorare il testo del terminale.

Questi codici permettono di rendere l'interfaccia più gradevole esteticamente. La sintassi è: \033[ (sequenza escape) + 1 (grassetto) + numero colore + m (terminatore)
