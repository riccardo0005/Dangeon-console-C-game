\doxysubsection{Detailed Description}
Definizioni di codici ANSI per colorare il testo del terminale. 

Questi codici permettono di rendere l\textquotesingle{}interfaccia più gradevole esteticamente. La sintassi è\+: \textbackslash{}033\mbox{[} (sequenza escape) + 1 (grassetto) + numero colore + m (terminatore)
